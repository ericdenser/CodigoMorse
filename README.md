========================================================
Como Executar:

Baixar o MinGW-w64 que inclui o GCC
Adicione o compilador ao PATH do Windows
Instale a Extensão C/C++ no VS Code
Criar arquivo .c
Clicar para executar como C/C++: Compile & Run
